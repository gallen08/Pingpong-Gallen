# Deskripsi
Proyek ini adalah game ping -pong sederhana berbasis Python menggunakan pustaka Pygame. Game ini dibuat sebagai latihan pemrograman Python untuk memahami konsep sprite, event-handling, dan logika permainan.

Dua pemain dapat bermain bersama di satu komputer. Pemain menggerakan raket untuk memantulkan bola. Pemain yg gagal memantulkan bola akan di nyatakan kalah.

# Fitur GAme
- 2 pemain (Kontrol Keyboard)
- Bola bergerak otomatis dengan pantulan realistis
- Aturan menang - kalah otomatis
- tampilan sederhana namun sudah di kembangkan

  # kontrol pemain
  Pemain 1 (Raket kiri)
  - W = niaik
  - S = turun
 
  Pemain 2 (raket kanan)
  UP Arrow = NAik
  Down arrow = turun

  2. Instal Pygame

     git clone https:/github.com/gallen08/Pingpong-Gallen
